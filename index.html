brain
javascript neural network library

Brain is a lightweight client-side and server-side(commonJS) javascript library for neural networks.

Brain implements the standard feedforward multi-layer perceptron neural network trained through the backpropagation algorithm.

It supports hashed (dynamic) input and output, JSON serialization, and options for tweaking the number of hidden layers/nodes and other network parameters.

what to use it for
There's lots of literature on what classes of problems neural networks are good at solving. One kind of problem that this sort of neural network is good at solving is classification problems (where the classes are known ahead of time).
what it can't be used for
This library in particular doesn't have a database component, so it's no good for building and training huge neural networks with thousands of different inputs. It's also not suited for real-time concurrent training or anything of that sort.
examples
Training the exclusive-or bitwise operation: 
  var data = [{input: [0, 0], target: [0]},
              {input: [0, 1], target: [1]},
              {input: [1, 0], target: [1]},
              {input: [1, 1], target: [0]}];

  var net = new NeuralNetwork();
  net.train(data);
  var output = net.run([1, 0]);
Serializing the network and plugging it in somewhere else: 
  var serialized = net.toJSON();

  var net2 = new NeuralNetwork({ json: serialized});
  net2.run([1, 0]);
Using a hashes instead of arrays for input and output: 
  var data = [{input: {red: 0, green: 0, blue: 1},    target: {blue: 1}},
              {input: {red: 1, green: 0.65, blue: 0}, target: {orange: 1}},
              {input: {red: 0, green: 1, blue: 0},    target: {green: 1}}];

  var net = new NeuralNetwork();
  net.train(data);
  var output = net.run({red: 1, green: 1, blue: 0}); // {"blue":0.009,"orange":0.981,"green":0.019}

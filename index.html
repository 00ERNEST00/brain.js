brain
javascript neural network library

brain is a lightweight JavaScript library for neural networks. It implements the standard feedforward multi-layer perceptron neural network trained with backpropagation.

An example, giving a color a label based on its rgb values: 

var net = new NeuralNetwork();

net.train([{input: {r:1, g:0.65, b:0},  output: {orange: 1}},
           {input: {r:0, g:0.54, b:0},  output: {green: 1}},
           {input: {r:0.6, g:1, b:0.5}, output: {green: 1}},
           {input: {r:0.67, g:0, b:1},  output: {purple: 1}}]);

var output = net.run({r:1, g:1, b:0});
the output would be something like {"orange":0.96,"green":0.18,"purple":0.00} because yellow is most similar to orange.

Download
To use this in the browser, download the latest brain.js. To use this as a commonJS module or server-side, checkout or download the code from github.
API
var net = new NeuralNetwork(options)
NeuralNetwork takes an optional argument that is a json object that can have members:
hidden - an array representing the number of nodes per hidden layer. e.g. [3, 4]. The default is one hidden layer, with a variable number of nodes.
growthRate - the rate at which to grow the hidden layer with respect to the size of input layer. The default is 0.5 (the hidden layer stays 50% the size of the input layer). This option does not apply if the hidden option was specified or the network was loaded in from json.
learningRate - the rate at which the network learns from training. Default is 0.5. The higher the number, the faster the training will go, but the risk of the network failing to train correctly increases.
net.train(data, iterations, threshold)
train trains the network on array of training data, where each item in the training data is a json object with members input and output. input and output can be either an array or json object of values from 0 to 1. e.g. 
    var data = [{input: [0.9, 0.4, 0], output: {orange: 1.0}},
                {input: [1.0, 1.0, 0], output: {yellow: 1.0}}]
The maximum number of training iterations and the error threshold to train to can also be specified. By default it will train for 20000 iterations or until the error drops below 0.01.
var output = net.run(input)
run runs the input on the neural network. the input should be an array or json object of values from 0 to 1. e.g. {a: 0.45, b: 0.02} or [0.45, 1.0]
var json = net.toJSON()
toJSON returns a JSON representation of the network.
net.fromJSON(json)
fromJSON loads a network from JSON. It will replace all the current layers and weights of the network.
var output = net.toFunction()(input)
toFunction returns a standalone function that mimics the run function of the network. Useful with toString() for training a network offline.
Demo
This webpage trains a network to recognize whether to use black or white text over a given background color. It is purely client-side, training the network on a Worker thread and using the JSON serializing function to pass the trained network back to the main thread. You can see the code for the site on brain's github.

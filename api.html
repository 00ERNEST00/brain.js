brain API
var net = new NeuralNetwork(options)
NeuralNetwork takes an optional argument that is a json object that can have members:
hidden - an array representing the number of hidden layers in the network and number of nodes per hidden layer. e.g. [3, 4]
growthRate - the rate at which to grow the hidden layer with respect to the size of input layer. The default is 0.4 (the hidden layer stays 40% the size of the input layer). This option does not apply if the hidden option was specified or the network was loaded in from json.
learningRate - the rate at which the network learns from training. Default is 0.5. The higher the number, the faster the training will go, but the risk of the network getting 'stuck' on local minima increases.
net.train(data, iterations, errorThresh)
train takes an array of training data, where each item is a json object with members input and target. input and output can be either an array or json object of values from 0 to 1. e.g. 
var data = [{input: {r: 0.9, g: 0.4, b: 0}, target: {orange: 1.0}},
            {input: {r: 1.0, g: 1.0, b: 0}, target: {yellow: 1.0}}]
the maximum number of training iterations and the error threshold to train to can also be specified. It will train until either the max number of iterations has been reached or until the average error dips below the error threshold. By default it will train for 20000 iterations or until the error drops below 0.01.
var output = net.run(input)
run runs the neural network on the given input. the input should be an array or json object of values from 0 to 1. e.g. {a: 0.45, b: 0.02} or [0.45, 1.0]
var json = net.toJSON()
toJSON returns a JSON serialization of the network.
net.fromJSON(json)
fromJSON loads a serialized network. It will replace all the layers/weights that the network currently has.
var run = net.toFunction()
toFunction returns a standalone function that mimics the run function of the network.
